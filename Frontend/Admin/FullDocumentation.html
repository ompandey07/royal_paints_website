Frequently Asked Questions

How do I reset my admin password?

Currently, the system uses default credentials (admin@tvs.com / admin@1200). To change the password, you'll need to access the Django admin interface or contact the system administrator for password reset functionality.

Can I import customers with different column names?

No, the system requires exact column names: "FULL NAME", "CONTACT NUMBER", "ADDRESS", and "EMAIL". Always download the latest template to ensure proper formatting.

What happens to old images when I update a blog post?

The system automatically deletes old images when you upload new ones for blog posts. This helps keep storage clean and prevents file accumulation.

How many SMS templates can be active at once?

Only one SMS template can be selected/active at a time. When you select a new template, the system will ask for confirmation if another template is already active.

Can I undo a customer data import?

There's no automatic undo feature. It's recommended to export current data before importing new customers, so you can restore if needed. Always test with small batches first.

Why are some customers showing as duplicates?

The system identifies duplicates based on contact numbers. If a customer with the same phone number already exists, you'll be prompted to choose whether to replace the existing record or ignore the duplicate.
